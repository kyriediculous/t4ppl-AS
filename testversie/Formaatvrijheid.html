Formaat vrijheid

Wat?

Een strategie voor de afhandeling van de input waarbij meerdere formaten mogelijk zijn.

Hulp bieden aan de gebruiker zodat die de input op de juiste manier vormgeeft.


Wanneer gebruiken?

Bij alle input die aan bepaalde vormvereisten moet voldoen en waarbij verschillende gebruiken zijn. In de USA wordt een datum anders weergegeven dan in Europa. 15 maart 2014 is in Europa 15/03/2014 en in de USA 03/15/2014.


Waarom?

De gebruiker heeft bepaalde gewoonten om gegevens vorm te geven. Het systeem moet zoveel mogelijk verschillende gewoonten ondersteunen zodat de gebruiker het systeem vlot kan gebruiken.


Richtlijnen en regels

    • Maak een lijst van courante formaten voor de gegevens;
    • Geef bij de input aan welk formaat u verwacht;
    • Gebruik controls die de input foutveilig maken (bv een dropdown voor dagen en maanden in een datum, een datapicker, een timepicker , selectie...);
    • Controleer de input, formaat aangeven maakt controle niet overbodig, het is een hulpmiddel, geen garantie.


Voorbeelden


Systeem leidt type input af

De gebruiker moet hier de postcode ingeven, de code van de luchthaven of de regio. Wat hij ingeeft speelt geen rol, het systeem kan ermee overweg.

Daarnaast krijgt hij ook een zoekmogelijkheid om de input te selecteren.
